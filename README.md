# flutter

Install flutter:
```bash
brew install --cask flutter
brew install --cask android-platform-tools
```

Create a new flutter project:
```bash
flutter create \
  --org com.shubhamtatvamasi \
  --project-name flutter_bounce .
```

List all the devices:
```bash
flutter devices
```

Start the app on your device:
```bash
flutter run -d I2220
```
